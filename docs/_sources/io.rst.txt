Data I/O
========

Save & Load
-----------
Pypianoroll supports efficient utilities for saving and loading
:class:`pypianoroll.Multitrack` object. The piano-roll will be first converted
to instances of `scipy.sparse.csc_matrix` and then stored in a .npz file.

.. autofunction :: save
.. autofunction :: load

Parse
-----
Pypianoroll currently supports parsing from MIDI files. Supports for MusicXML
and ABC format are in the plan.

.. autofunction :: parse
